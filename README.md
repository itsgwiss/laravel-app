# 🐝 Grace Integrative API Prototype

A Laravel 12 application integrating **Authentication**, **OTP Verification**, **Mailbox**, and **AI Chatbot** modules.

---

## Stack

| Layer     | Technology                              |
|-----------|-----------------------------------------|
| Backend   | PHP 8.2 + Laravel 12                    |
| Frontend  | Blade + Tailwind CSS + Alpine.js        |
| Auth      | Laravel Breeze (email/password)         |
| OTP       | Cache-based 6-digit codes (email + SMS) |
| Mailbox   | Eloquent `messages` table               |
| AI Chat   | Anthropic Claude API (+ rule-based fallback) |
| DB        | SQLite (default) or MySQL/PostgreSQL    |

---

## Quick Start

```bash
# 1. Clone and install
git clone https://github.com/ckentdev/tcc-integrative-api-prototype.git
cd tcc-integrative-api-prototype

# 2. Install PHP dependencies
composer install

# 3. Copy environment file and generate key
cp .env.example .env
php artisan key:generate

# 4. Run migrations
php artisan migrate

# 5. Install and build frontend assets
npm install
npm run build

# 6. Start the dev server
composer run dev
```

Open http://localhost:8000

## Environment Variables

Edit `.env` after copying from `.env.example`:

```env
# Required
APP_KEY=          # generated by key:generate

# Optional — enables real AI responses in the chatbot
ANTHROPIC_API_KEY=sk-ant-...

# Optional — enables real SMS delivery for OTP
TWILIO_SID=
TWILIO_TOKEN=
TWILIO_FROM=+1xxxxxxxxxx

# Mail (use log driver for dev — OTP codes appear in storage/logs/laravel.log)
MAIL_MAILER=log
```

## Module Overview

### 🔐 Authentication
- Register, Login, Logout via Laravel Breeze
- Email verification required before accessing app
- Password reset via email link
- Views: `resources/views/auth/`

### 🔢 OTP Verification
- **Email OTP**: sends 6-digit code via `Mail`, stored in Cache (10 min TTL)
- **SMS OTP**: stub ready for Twilio/Vonage integration
- Auto-advancing 6-input UI with paste support
- Routes: `POST /otp/send-email`, `POST /otp/send-sms`, `POST /otp/verify`
- Controller: `app/Http/Controllers/OTPController.php`

> **Dev tip**: OTP codes are logged to `storage/logs/laravel.log` when `MAIL_MAILER=log`

### 📬 Mailbox
- Inbox and Sent views for authenticated user
- Compose modal with AJAX send (no page reload)
- Messages stored in `messages` table
- Real email delivery attempted (falls back gracefully)
- Routes: `GET /mailbox`, `POST /mailbox/send`
- Controller: `app/Http/Controllers/MailboxController.php`

### 🤖 AI Chatbot (Grace)
- Powered by Anthropic Claude API (`claude-haiku-4-5-20251001`)
- Falls back to rule-based replies if no API key configured
- Typing indicator animation
- AJAX-based, no page reloads
- Route: `POST /chatbot/message`
- Controller: `app/Http/Controllers/ChatbotController.php`

## Adding SMS (Twilio)

1. Install the SDK: `composer require twilio/sdk`
2. Set `TWILIO_SID`, `TWILIO_TOKEN`, `TWILIO_FROM` in `.env`
3. Uncomment the Twilio block in `OTPController::sendSms()`
